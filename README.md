🚦 Road Accident Analysis Dashboard (Power BI)

This project presents an interactive Power BI dashboard designed to analyze road accident data and uncover key patterns related to accident frequency, severity, and contributing factors. The goal is to provide actionable insights that can support traffic safety improvements and data-driven decision-making.

🔍 Key Features
Interactive dashboard with KPIs, trend analysis, and geographic insights
Analysis of accident frequency, vehicle types, and high-risk zones
Identification of peak accident times and major contributing factors
Use of DAX measures for advanced metrics like fatality rate and trend analysis
Data transformation and modeling using Power Query
📊 Insights Generated
High-risk areas with maximum accident occurrences
Time-based trends (daily, monthly patterns)
Impact of different vehicle types on accident severity
Key factors contributing to road accidents
🛠️ Tech Stack
Power BI
DAX (Data Analysis Expressions)
Power Query
